# ☀️ Агрегатор погодных данных

## 📖 О проекте
`weather_aggregator` — это проект, представляющий собой Python-скрипт, который получает данные о погоде из внешнего REST API, сохраняет их в локальную базу данных SQLite, вычисляет ключевые метрики и визуализирует результаты с помощью графика.

## ⚙️ Примеры работы
**Вывод консольного отчёта:**

При запуске скрипта в консоли отображается следующий аналитический отчёт:
```bash
--- Аналитический отчет ---
1. Средняя максимальная температура: 15.2 °C
2. Средняя минимальная температура: 5.8 °C
3. Самый теплый день: 2026-06-01 (25.0 °C)
4. Самый холодный день: 2026-05-30 (2.1 °C)
5. Размах температур: 22.9 °C
```

- **Визуализация данных:**
После вывода текста автоматически открывается окно с графиком:

**1. Линейный график** "Прогноз погоды", отображающий динамику максимальной и минимальной температуры за последние 7 дней.

## 💡 Использование
**Шаг 1: Установка зависимостей**

Установите необходимые библиотеки из файла requirements.txt:

```bash
pip install -r requirements.txt
```
Или установите вручную:
```bash
pip install requests matplotlib pandas
```

**Шаг 2: Запуск программы**

Запустите скрипт из терминала в корневой папке проекта:

```bash
python src/main.py
```

## 🧠 Логика анализа
**Алгоритм работы скрипта:**

- Выполнение GET-запроса к публичному API погоды (Open-Meteo) для получения данных за последние 7 дней.
- Парсинг полученного JSON-ответа.
- Сохранение данных (дата, макс. температура, мин. температура) в таблицу weather_log базы данных SQLite.
- Вычисление аналитических метрик на основе данных из БД.
- Визуализация результатов с помощью библиотеки Matplotlib.

**Особенности реализации:**

- Для расчёта средних значений используются SQL-функции `AVG()`.
- Поиск самого теплого и холодного дня осуществляется с помощью сортировки `ORDER BY ... DESC/ASC LIMIT 1`.
- Размах температур вычисляется как разница между глобальным максимумом и минимумом `(MAX() - MIN())`.
- Визуализация представляет собой линейный график, где по оси X отложены даты, а по оси Y — значения температуры.

## 🛠️ Технологии
- **Python 3.8+**
- requests: библиотека для выполнения HTTP-запросов к API.
- sqlite3: стандартная библиотека для работы с локальной базой данных SQLite.
- matplotlib: библиотека для построения графиков и визуализации данных.
- pandas: библиотека для анализа данных, используется для удобства построения графика.
- json: стандартная библиотека для парсинга JSON-данных.
